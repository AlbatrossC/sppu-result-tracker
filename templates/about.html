About this tracker

A cleaner way to follow SPPU result declarations

SPPU Result Tracker is an independent project that watches the publicly available result information from Savitribai Phule Pune University and presents it in a simple, student-friendly format.

Independent Project

This website is not affiliated with SPPU. It is not endorsed by, maintained by, sponsored by, or officially connected with Savitribai Phule Pune University.

For any official academic decision, verification, marksheet, eligibility, revaluation, or certificate-related step, always refer to the official SPPU website or your college.

Why This Exists

Students often need to check whether a result has been declared, but the official result page can be inconvenient to scan repeatedly. This tracker keeps the public information focused on the two details students usually need first: the course name and the result date.

The goal is not to replace the official source. The goal is to make discovery faster, cleaner, and easier to follow.

What It Shows

  • Declared result entries found on the public SPPU result page.
  • The course name exactly as tracked by the system.
  • The result date associated with that course.
  • The latest time this tracker successfully checked for updates.

What It Does Not Show

  • Individual student marks or seat numbers.
  • Private student data of any kind.
  • Official confirmation of pass, fail, revaluation, or eligibility status.
  • College-specific notices that are not present in the tracked source.

Discord Result Alerts

The Discord server is for quick notifications. When the tracker detects a newly declared result, it sends a simple alert so students do not have to keep refreshing the page.

Alert wording New declarations are announced as "Result Declared" so the message is clear and user-facing.
Delivery timing Alerts depend on the tracker run and Discord availability, so they may not always be instant to the second.

Accuracy And Limitations

The tracker tries to mirror public result declarations accurately, but it can be affected by source-page changes, network issues, hosting delays, or temporary service outages.

If something looks unusual, treat this website as a helpful alerting tool and cross-check the information with the official SPPU source before relying on it.

Privacy

This tracker is designed around public result declaration entries. It does not ask for roll numbers, seat numbers, names, phone numbers, passwords, or other private student information.

Open Source

The source code is available on GitHub so the project can be inspected and improved openly.
